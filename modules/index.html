  • 1. How to Set Up Your Raspberry Pi

    This lesson covers how to set up your Raspberry Pi computer. The Raspberry Pi is special because it's a hackable computer; you can change things, download lots of free software, and use it in all sorts of interesting ways.

  • 2. How Control Your Raspberry Pi

    This lesson covers the basics of how to use your Raspberry Pi. An RPi is a computer, so you can use it just like any other! It also has some special features that might be new to you.
